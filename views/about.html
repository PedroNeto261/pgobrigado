About

Profile

Nossa empresa surgiu a 3 meses quando seus fundadores Andre Luna, Thiago Felix, Pedro Neto, Marcelo Jácome, Janaina Menezes e Beta Damaso perceberam que os moradores da cidade Smith tinham muitos problemas para adquirir produtos esportivos de qualidade e ficar por dentro de todas as novidades do mundo esportivo




Projeto Hexa

Projeto HEXA, temos parceirias com escolinhas de futebol por todo o Brasil e nossa empresa está empenhada em revelar novos talentos do futebol, para isso, inicamos um projeto social com o projeto Hexa, o projeto consiste em apresentar o esporte para crianças carentes, queremos alcançar o maior numero de crianças possiveis e mostrar que o esporte consegue mudar vidas. O Sport Center Soul quer revelar os novos talentos para trazer o nosso Hexa, estamos cansados de não conseguir o hexa então para isso temos esse fantástico e divertido projeto.

Nossa atitude

Nós acreditamos que o esporte pode mudar vidas, sendo assim, temos como objetivo levar o esporte para todos os lugares, queremos alcançar o mundo todo. Para isso acontecer precisamos iniciar de alguma forma, então queremos começar a iniciar projetos sociais para oferecer gratuitamente a prática de esportes para quem nunca teve a oportunidade de praticar algo.

Melhor postura

O esporte pode mudar vidas, esse é nosso slogan, por isso temos o projeto TRAPEZIO DESCENDENTE, esse projeto leva academia para todos, esse projeto nasceu devido ao alto número de jovens com problema de postura, esses problemas estavam prejudicando o seu desenvolvimento corporal então fizemos uma parceria com o grande Filipe Smith personal internacional para que ele possa auxilar nossos jovens com treinos diarios realizados durante lives no nosso instagram.
